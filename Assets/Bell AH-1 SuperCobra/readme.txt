Use "ArmatureMainRotor" and "ArmatureTailRotor" GameObjects to rotate blades
Use "EmptyRadar", "EmptyTurret" and "EmptyCanon"(EmptyTurret's child) to rotate weapons system.

Doors and rotors are driven by the animator. (Open,Close,Bend,Pitch)

Use "Custom PSD" to create your own texture

It's highly recommended to code your own script, provided scripts are for testing purpose only.

Control 
Shift : up
Ctrl : down
arrow : move
N : Start rotors
B : Shut down rotors
Y : Toogle Pilot Door
U : Toogle Gunner Door
T : Lock target
G : Unlock target
